# Dev-Tools-Domination
Extensive lesson in developer tools - Done as part of JavaScript30, 30 day vanilla JS coding challenge

Dev tools used:
  * console.log()
  * console.warn()
  * console.error()
  * console.info()
  * console.assert()
  * console.clear()
  * console.dir()
  * console.group() / console.groupEnd()
  * console.count()
  * console.time()
  * console.table()
  * Interpolation using %s and ${}
  * Styling using %c
